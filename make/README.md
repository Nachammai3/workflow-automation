# Make.com Automation Project

## Telegram Watch Updates Automation

This project demonstrates an automation workflow built using **Make.com**.
The automation watches incoming Telegram bot updates and processes them using an AI agent.

### Workflow Steps
1. Telegram Bot monitors incoming messages
2. Make AI Agent analyzes the message content
3. Scenario processes the data and returns structured output

### Tools & Technologies
- Make.com
- Telegram Bot
- AI Agents
- Scenario-based Automation

### Use Case
- Automated message monitoring
- AI-powered data processing
- No-code workflow automation
