# n8n Workflow Automation

## AI-powered Content Automation Workflow

This project demonstrates an automation workflow built using **n8n** to process data using AI models and automate content posting.

### Workflow Overview
- Triggered by updates from Google Sheets
- Processes content using LLM chains
- Uses OpenAI Chat Model for summarization
- Automatically posts processed content to LinkedIn and X (Twitter)

### Tools & Technologies
- n8n
- OpenAI / LLM Chains
- Google Sheets
- LinkedIn API
- X (Twitter) API

### Use Cases
- Automated content generation
- AI-based text summarization
- Social media automation
- No-code / low-code workflow automation
